# 居家健身训练指南

> 适合久坐办公、体重正常但核心薄弱的人群。无需健身房，居家即可执行。
>
> 📅 践行记录：2026-02-14 至 2026-06-12，累计运动 **90 天**。

---

## 一、先做身体评估

在开始训练前，先判断自己的体型类型：

### 1.1 你属于哪种体型？

| 类型 | 特征 | 对策 |
|------|------|------|
| **瘦胖子** | BMI 正常（18.5-23.9），但坐着腹部有一圈软肉，肌肉量偏低 | 身体重组：增肌+减脂同时进行 |
| **偏瘦型** | BMI < 18.5，整体偏瘦 | 增肌为主，热量盈余 |
| **超重型** | BMI > 24，全身脂肪偏多 | 减脂为主，控制热量缺口 |

### 1.2 久坐人群的问题根源

**腹横肌松弛**
腹横肌是深层核心肌肉，像一条天然束腰带。久坐导致它长期不发力，坐着时无法收紧兜住内脏和脂肪，肚子自然堆出来。

**骨盆前倾或坐姿不良**
很多久坐男性有骨盆前倾（站立时腰前凸、小腹前顶）。坐着时骨盆后倒，腹部软组织被挤压折叠。

**肌肉量偏低**
肌肉少则基础代谢低，腹部脂肪更容易储存，且皮肤支撑力弱。

## 二、核心训练（收紧腰腹）

### 2.1 真空腹 — 最有效的腰腹收紧动作

这是收紧腹横肌最直接的动作，能减少坐着时的腹部突出。

**正确做法（四足跪姿版，适合初学者）：**
1. 四肢跪地，手在肩正下方，膝盖在髋正下方
2. 彻底呼气，感觉肺里空气排空
3. 用力将肚脐向脊柱方向**吸**，想象把腹部"拉空"成一个真空状态
4. **保持正常呼吸**（用胸式呼吸，腹部保持内收），不要憋气
5. 保持 15-30 秒，放松。重复 5-10 次

> 关键：不是"吸肚子"（那只是收缩腹直肌），而是收缩深层腹横肌。做对时，腰围会肉眼可见地变细一圈。

**建议频率**：每天早晨空腹做 5 组

### 2.2 其他核心动作

| 动作 | 组数 | 次数/时长 | 作用 |
|------|------|-----------|------|
| 平板支撑 | 3 组 | 30-60 秒 | 整体核心稳定 |
| 死虫式 | 3 组 | 10 次/侧 | 腹横肌激活 |
| 鸟狗式 | 3 组 | 10 次/侧 | 核心平衡+脊柱稳定 |
| 臀桥 | 3 组 | 15 次 | 激活臀肌，纠正骨盆前倾 |

## 三、背部训练（改善体态）

### 3.1 为什么一定要练背？

- **改善体态**：强化背部可以拉直肩膀，改善圆肩驼背
- **视觉上显腰细**：背部练宽，形成"倒三角"身材
- **提升全身力量**：几乎所有 pulling（拉）的动作都需要背部参与

### 3.2 背部主要肌肉

| 肌肉 | 功能 | 训练动作 |
|------|------|---------|
| **背阔肌** | 决定背部宽度 | 高位下拉、引体向上 |
| **斜方肌（中下束）+ 菱形肌** | 决定背部厚度 | 划船类动作 |
| **竖脊肌** | 下背部稳定 | 硬拉、超人式 |

### 3.3 居家背部训练动作

**宽度训练（下拉类）：**
- **引体向上**：练背的王牌动作。如无法完成，可用弹力带辅助或做负重引体
- **门框划船**：双手抓住门框两侧，身体后倾，拉向胸部

**厚度训练（划船类）：**
- **弹力带划船**：将弹力带固定在前方，坐姿拉向腹部，顶峰夹紧肩胛骨
- **俯身哑铃/矿泉水瓶划船**：身体前倾，核心收紧，将重物沿大腿拉向下腹部

> 对新手来说，练背的核心是**找到感觉**，而不是上大重量。

## 四、全身力量训练计划

### 4.1 核心原则

- **不节食、不做长时间有氧**（避免掉肌肉）
- **以力量训练为主，有氧为辅**
- **渐进超负荷**：每周增加次数或组数

### 4.2 周计划模板

**周一、三、五（力量+核心）**
```
热身：猫牛式 10 次 + 臀桥 15 次
训练：
  - 深蹲 3×10 次
  - 俯卧撑 3×8 次（力竭前保留 2 个）
  - 弹力带划船 3×12 次
  - 平板支撑 3×30 秒
核心：
  - 死虫式 3×10 次/侧
  - 鸟狗式 3×10 次/侧
冷身：真空腹 5 组（每组保持 20 秒）
```

**周二、四（短时有氧+体态）**
```
快走 30 分钟（心率 ≈ 130）
或 开合跳/高抬腿（做 20 秒休 10 秒，循环 8 轮）
额外：靠墙站立 5 分钟
```

**每天必做**
```
早晨空腹：真空腹 5 组
久坐后：站起来做 10 次臀桥 + 深吸气收缩腹横肌 10 秒
睡前：拉伸髋屈肌（弓箭步拉伸，每侧 30 秒）
```

### 4.3 渐进超负荷方法（居家版）

| 动作 | 进阶方式 |
|------|---------|
| 深蹲 | 自重 → 高脚杯深蹲（抱重物）→ 分腿蹲 |
| 俯卧撑 | 跪姿 → 标准 → 脚抬高 |
| 划船 | 弹力带 → 装满水的背包 → 更重的水桶 |

每周增加总容量：比如本周深蹲共做 30 次（3×10），下周尝试 33 次（3×11）或增加一组。

## 五、饮食营养搭配

### 5.1 蛋白质目标

| 目标 | 摄入量 | 食物来源 |
|------|--------|---------|
| 增肌 | 1.6-2.0g/kg 体重 | 鸡蛋、鸡胸肉、鱼、豆腐、牛奶 |
| 维持 | 1.2-1.6g/kg 体重 | 同上，可减少份量 |

**参考**：57.5kg 体重的男性 → 每天 110-120g 蛋白质
≈ 4 个鸡蛋 + 200g 鸡胸肉/鱼 + 300ml 牛奶

### 5.2 碳水选择

**优先选择（慢碳水）：**
- 燕麦片（非即食）、糙米、红薯、玉米
- 全麦面包、荞麦面、藜麦

**需避免：**
- 即食麦片（甜、脆的）、白面包、白馒头
- 含糖饮料、果汁、风味酸奶

> 建议：一半主食换成粗粮，先吃蔬菜和蛋白质，最后吃主食。

### 5.3 牛奶怎么选

| 类型 | 推荐度 | 理由 |
|------|--------|------|
| **全脂纯牛奶** | ⭐⭐⭐ | 热量适中，口感好，利于吸收脂溶性维生素 |
| 低脂牛奶 | ⭐⭐ | 适合严格控制脂肪摄入时 |
| 脱脂牛奶 | ⭐ | 热量过低，不利于维持体重 |
| 风味牛奶 | ❌ | 添加大量糖，不推荐 |

> 每天建议 300-500ml，可提供 10-16g 优质蛋白。

## 六、运动记录方法

### 6.1 记录什么

建议建立运动日志，记录以下内容：
- **日期**：每天打卡，形成习惯
- **动作名称** + **组数/次数** + **时长/组**
- **完成情况**：✅ 完成 / ⚠️ 部分完成 / ❌ 未完成
- **身体感受**：肌肉酸痛程度、疲劳感

### 6.2 记录模板

```
## 周运动日志

| 日期 | 运动项目 | 完成情况 | 备注 |
|------|----------|----------|------|
| 周一 | 深蹲3×10 + 俯卧撑3×8 + 平板3×30s | ✅ | 感觉背部发力明显 |
| 周二 | 快走30分钟 | ✅ | 心率保持130 |
| 周三 | 深蹲3×12 + 俯卧撑3×10 + 平板3×40s | ✅ | 进阶，增加2次 |
| ... | ... | ... | ... |
```

### 6.3 周期计划示例

以 7 天为一个周期：
- **周期1**：适应动作、找到发力感
- **周期2**：增加次数，固定动作模式
- **周期3**：增加组数或缩短组间休息
- **周期4**：尝试进阶版本（如标准→负重）

每 4 周评估一次效果（腰围、体重、力量变化），调整下一阶段计划。

## 七、常见误区

| 误区 | 表现 | 正确做法 |
|------|------|---------|
| **疯狂做仰卧起坐减肚子** | 腰更粗，坐着肉还在 | 坚持真空腹（减少腰围）+ 全身力量训练（提高代谢） |
| **蛋白质吃不够** | 训练后恢复慢，肌肉不涨 | 每天吃到 1.6-2.0g/kg 体重 |
| **有氧时间太长** | 体重下降，但掉肌肉 | HIIT ≤15 分钟，或快走 ≤40 分钟 |
| **体态纠正腰酸痛** | 做臀桥或平板时腰疼 | 说明腹部没收紧。降低难度，提肛收腹 |
| **前 3 周看不到变化就放弃** | 肌肉储水期感觉肚子更胀 | 这是正常生理反应，坚持到第 4 周 |

---

## 八、总结：新手起步流程

```
第 1 周 → 掌握真空腹 + 找到核心发力感
第 2 周 → 加入全身力量训练（深蹲、俯卧撑、划船）
第 3 周 → 固定训练节奏，调整饮食蛋白质
第 4 周 → 评估效果，增加强度
第 5-8 周 → 渐进超负荷，巩固成果
```

> 训练不是越多越好，而是**持续 + 渐进 + 恢复**的平衡。每天 20-30 分钟，坚持 4 周以上才能看到明显变化。
