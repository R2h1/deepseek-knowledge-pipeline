# 实用工具集：创意与实现方案

> 好的工具解决一个具体问题，一个好用的工具让人用完即走，但下次还会回来。

---

## 一、效率工具

### 1.1 待办清单（Todo List）

**核心功能：**
- 添加/编辑/删除待办事项
- 标记完成/未完成
- 按优先级排序（高/中/低）
- 按日期分组（今天/本周/以后）
- 本地存储（localStorage）

**技术方案：**
- 纯前端，Vue 3 + localStorage
- 单页应用，无需后端
- 数据导出为 JSON/CSV

### 1.2 番茄钟

**核心功能：**
- 25 分钟专注计时
- 5 分钟休息计时
- 每 4 个番茄钟后长休息（15 分钟）
- 开始/暂停/重置
- 提示音通知

**技术方案：**
- 浏览器 Notification API 做推送
- Audio API 做提示音
- 统计每日完成番茄数

### 1.3 倒数日/倒计时

**核心功能：**
- 设置目标日期和名称
- 显示剩余天数/小时/分钟
- 支持多个倒数项目
- 本地存储

## 二、工具类

### 2.1 随机密码生成器

**核心功能：**
```
可选：
  ☑ 大写字母 (A-Z)
  ☑ 小写字母 (a-z)  
  ☑ 数字 (0-9)
  ☑ 特殊符号 (!@#$%^&*...)
  
选项：
  密码长度：8-32 位
  排除相似字符（如 1/l/O/0）
  一键复制到剪贴板
  密码强度指示器
```

**密码强度参考：**
```
8 位（大小写+数字）   → ⭐⭐ 几分钟破解
12 位（大小写+数字+符号）→ ⭐⭐⭐⭐ 数年破解
16 位（混合）         → ⭐⭐⭐⭐⭐ 几乎不可能破解
```

### 2.2 填字挑战

小游戏类工具，提供每日填字谜题。核心逻辑：
- 加载词库和提示
- 用户输入字母
- 自动校验正确性
- 显示提示和答案

### 2.3 亲戚称呼计算器

输入两个人的关系链，输出正确的称呼。

**示例：**
```
输入：爸爸的哥哥的儿子
输出：堂哥/堂弟

输入：妈妈的妹妹的女儿
输出：表姐/表妹
```

**核心规则库：**
```
父亲这边 → 堂亲（同姓）
母亲这边 → 表亲（不同姓）
年长 → 哥/姐
年幼 → 弟/妹
```

### 2.4 舒尔特表

注意力训练工具。一个 5×5 的表格，数字 1-25 乱序排列，用户按顺序点击。

```
核心指标：
  完成时间 → 注意力集中度判断
  
参考标准：
  < 16 秒 → 注意力优秀
  16-25 秒 → 正常
  > 25 秒 → 需要提高
```

### 2.5 每日菜单生成器

解决选择困难症的实用工具：

- **工作日模式**：附近外卖随机推荐（可排除不喜欢的）
- **周末模式**：根据冰箱存货推荐家常菜
- **个性化**：记录偏好，逐步优化推荐算法

## 三、格式转换工具

### 3.1 时间格式转换

```
输入：2026-06-13T10:30:00
输出：
  - 2026年6月13日 10:30
  - 2026-06-13 10:30:00
  - 06/13/2026
  - 时间戳：1747197000
  - 相对时间：3 小时前
```

### 3.2 枚举值转换

```javascript
// 输入：enum 键
"status_1" → "待处理"
"status_2" → "处理中"  
"status_3" → "已完成"

// 支持中英文互转、大小写转换、驼峰/下划线互转
```

## 四、平台构想：Conan 工具站

所有这些工具都可以聚合到一个平台中，核心设计：

### 4.1 首页布局

```
┌─────────────────────────────────┐
│  🔍 搜索工具                     │
├──────┬──────┬──────┬──────┬─────┤
│ 待办  │ 番茄钟 │ 密码  │ 倒数  │ ...  │
│ 清单  │       │ 生成器 │ 日    │      │
├──────┼──────┼──────┼──────┼─────┤
│ 填字  │ 称呼  │ 舒尔特│ 菜单  │ ...  │
│ 挑战  │ 计算器 │ 表   │ 生成器 │      │
├──────┼──────┼──────┼──────┼─────┤
│ 时间  │ 枚举  │ ...   │      │      │
│ 转换  │ 转换  │       │      │      │
└──────┴──────┴──────┴──────┴─────┘
```

### 4.2 工具页通用模板

每个工具都包含：
- **工具名称** + **简短描述**
- **核心功能区域**（输入/操作区）
- **结果展示区**
- **帮助提示**（用法说明）

### 4.3 技术栈建议

```
前端：Vue 3 + Composition API + TypeScript
UI 框架：shadcn-vue 或 Element Plus
状态管理：Pinia
路由：Vue Router
存储：localStorage（后期可加后端）
部署：静态页面（Vercel/Netlify/Cloudflare Pages）
```
