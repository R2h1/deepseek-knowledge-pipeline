## 一、Windows 技巧

### 1.1 Windows 11 常用快捷键

| 快捷键 | 作用 |
|--------|------|
| `Win + D` | 显示桌面（再按恢复） |
| `Win + E` | 打开文件资源管理器 |
| `Win + I` | 打开设置 |
| `Win + L` | 锁定电脑 |
| `Win + V` | 打开剪贴板历史（比 Ctrl+V 更强） |
| `Win + .` | 打开 emoji 选择器 |
| `Win + Shift + S` | 屏幕截图（自带截图工具） |
| `Win + 数字键` | 打开任务栏对应位置的程序 |
| `Ctrl + Shift + Esc` | 直接打开任务管理器 |
| `Alt + Tab` | 切换窗口 |
| `Win + Tab` | 任务视图（显示所有桌面和窗口） |

### 1.2 键盘重启电脑

如果电脑卡死，无法正常点击重启：

```
方法 1：Ctrl + Alt + Delete → 右下角电源按钮 → 重启
方法 2：长按电源键 10 秒强制关机 → 再按一次开机
方法 3：Alt + F4（桌面时）→ 选择"重新启动"
```

> ⚠️ 方法 2 是最后手段，频繁强制关机可能损坏系统。

### 1.3 VSCode 小键盘数字键失灵

在 VSCode 中，小键盘（Num Pad）的数字键有时不能正常输入数字，而是触发光标移动等操作。

**原因**：VSCode 的编辑器设置中，小键盘被映射为其他功能。

**解决办法：**

```
打开设置（Ctrl + ,）
搜索 "editor.numericInput" 或 "numPad"
找到 "Editor: Handle Numeric Input" 选项
取消勾选 → 重启 VSCode
```

## 二、手机摄影技巧

### 2.1 手机摄影的核心原则

手机摄影的劣势是传感器小、弱光表现差，但优势是**随身携带、随手拍**。

| 原则 | 说明 |
|------|------|
| **光线就是一切** | 手机摄影最怕暗光，找光比用任何技巧都重要 |
| **靠近，再靠近** | 手机没有光学变焦，数码变焦会损失画质，物理靠近最好 |
| **稳比什么都重要** | 双手握持，手肘夹紧身体，或靠在墙上 |
| **后期不是作弊** | 手机的算法本身就是"后期"，用 Snapseed、Lightroom 简单调一下效果会好很多 |

### 2.2 三种常用场景的设置

**人像模式：**
- 保持 1-2 米距离
- 确保主体和背景有明显层次
- 后期可以调整虚化程度

**夜景模式：**
- 手持要尽量稳，最好找支撑
- 按下快门后保持静止 2-3 秒
- 太暗的场景建议用"专业模式"提高 ISO

**专业模式参数参考：**
```
ISO：尽量低（100-400），画面更干净
快门：1/60s 以上手持，低于此需三脚架
对焦：拍人用 AF，拍微距用手动 MF
白平衡：日光（晴天）/ 阴天（多云）/ 白炽灯（室内）
```

## 三、宽带办理与手机网络

### 3.1 异地手机卡如何装宽带？

如果你人在外地，但手机卡归属地不在本地，装宽带需要：

| 方案 | 操作 | 费用 |
|------|------|------|
| **融合套餐** | 将手机卡归属地迁移到当地（部分运营商支持） | 免费或少量费用 |
| **单独办宽带** | 用身份证在当地直接新开宽带（不绑定手机） | 通常比融合套餐贵 10-20 元/月 |
| **携号转网** | 将号码转到当地运营商后办理融合套餐 | 免费，需满足转网条件 |

> 江苏移动用户在深圳装宽带：建议直接办深圳移动的**单宽带**，或者把号码办理"省内携号"到深圳移动。

### 3.2 手机尺寸选择（速查）

| 手型 | 推荐屏幕尺寸 |
|------|-------------|
| 小手（掌宽 < 8cm） | 5.4"-6.1" |
| 中等手（掌宽 8-9cm） | 6.1"-6.5" |
| 大手（掌宽 > 9cm） | 6.5"+ |

## 四、实用工具

### 4.1 ripgrep（rg）是什么？

ripgrep（rg）是一个比传统 grep 快得多的文本搜索工具，用 Rust 编写。

| 对比 | grep | ripgrep（rg） |
|------|------|--------------|
| 速度 | 慢（递归搜索大目录时明显） | 快 5-10 倍 |
| 默认行为 | 不递归搜索 | 默认递归 |
| 忽略文件 | 不会自动忽略 .gitignore | 自动尊重 .gitignore |
| 编码支持 | 需要额外配置 | 自动处理 UTF-8 |

**Windows 下安装：**
```
方法 1（推荐）：winget install BurntSushi.ripgrep
方法 2（手动）：从 GitHub 下载 .zip，解压到 PATH 目录
```

### 4.2 清远能见度低的原因

清远偶尔出现能见度低的现象，主要原因：

- **地形**：清远地处广东北部山区，山谷地形容易形成雾气
- **湿度**：珠三角地区湿度大，早晨和雨后容易起雾
- **工业与交通**：珠三角工业排放+汽车尾气，在特定气象条件下形成雾霾
- **季节**：冬春季节逆温层明显，污染物不易扩散

> 能见度低通常是**雾**（水汽）和**霾**（颗粒物）的混合物。关注空气质量指数（AQI）比单纯看能见度更能判断是否为污染。
