# 选购指南系列

> 买东西不是越贵越好，而是**适合自己**最好。以下四份选购指南覆盖日常消费到奢侈品的不同维度。

---

## 一、手机屏幕尺寸选择指南

### 1.1 主流屏幕尺寸分类

| 类型 | 尺寸范围 | 代表机型 |
|------|---------|---------|
| 小屏 | 5.4"-5.8" | iPhone 13 mini、Unihertz Jelly |
| 标准 | 6.1"-6.3" | iPhone 15/16 Pro、Galaxy S |
| 大屏 | 6.5"-6.7" | iPhone Pro Max、小米 14 Ultra |
| 巨屏 | 7"+ | 折叠屏展开态、iPad mini |

### 1.2 如何根据手型选择

一个简单的测试：**单手握持时，大拇指能否覆盖屏幕对角的 70% 区域？**

| 测试结果 | 建议 |
|---------|------|
| 轻松覆盖 | 可上大屏（6.5"+），兼顾视觉和操作 |
| 勉强覆盖 | 标准屏（6.1"-6.3"）最保险 |
| 够不到 | 小屏（6.0"以下）或选择薄机身机型 |

### 1.3 不同场景的屏幕偏好

| 使用场景 | 推荐尺寸 | 原因 |
|---------|---------|------|
| 刷视频/看电影 | 6.5"+ | 视觉沉浸感更强 |
| 单手操作/通勤 | 6.1"-6.3" | 握持稳定，不易滑落 |
| 阅读/文档 | 6.5"+或折叠屏 | 减少翻页频率 |
| 游戏 | 6.3"-6.7" | 兼顾操控区域和视野 |
| 运动携带 | 6.0"以下 | 轻便、易收纳 |

### 1.4 屏幕素质核心指标

除了尺寸，还要关注：

- **分辨率**：至少 FHD+（2400×1080），2K 更佳
- **刷新率**：120Hz 已成为主流，流畅度明显优于 60Hz
- **亮度**：户外使用需要 1000nit 以上峰值亮度
- **调光**：DC 调光或高频 PWM 调光（>1920Hz）对眼睛更友好

## 二、隐形眼镜 vs 普通眼镜对比

### 2.1 对比表格

| 维度 | 隐形眼镜 | 普通框架眼镜 |
|------|---------|------------|
| **视野** | 无边框限制，视野完整 | 有镜框边缘，视野受限 |
| **运动** | 方便，不易滑落 | 可能滑落，起雾 |
| **温差** | 不受温差影响 | 冬天进屋起雾 |
| **护理** | 每日摘戴、清洁，需定期更换 | 偶尔擦拭即可 |
| **干眼** | 长时间佩戴容易眼干 | 无此问题 |
| **成本** | 年消耗约 1,000-3,000 元 | 一副用 1-3 年 |
| **矫正范围** | 散光需要定制，价格更高 | 所有度数均适用 |

### 2.2 隐形眼镜类型选择

| 类型 | 更换周期 | 特点 |
|------|---------|------|
| **日抛** | 每天换 | 最卫生，无需护理，适合偶尔佩戴 |
| **月抛** | 每月换 | 性价比最高，需要每日护理 |
| **年抛** | 每年换 | 最经济，但蛋白沉积风险高，不建议 |

> 对新手来说，**日抛**是最安全的选择。多花的钱买的是眼睛健康。

### 2.3 佩戴注意事项

```
✅ 戴前洗手，用无香皂、无油分
✅ 先戴镜片再化妆，先摘镜片再卸妆
✅ 每天佩戴不超过 10 小时
✅ 感到干涩时用隐形专用润眼液

❌ 不要戴着睡觉（除非医生允许的长戴型）
❌ 不要用水冲洗镜片（用护理液）
❌ 不要超期使用（日抛当天扔，月抛到期换）
❌ 不要共用镜片
```

## 三、初学者吉他选购指南

### 3.1 新手避坑要点

| 陷阱 | 说明 | 建议 |
|------|------|------|
| **贪图低价** | 500 元以下吉他音准差、弦距高 | 入门预算建议 800-1500 元 |
| **只看颜值** | 花哨漆面掩盖木材缺陷 | 关注手感参数 |
| **唯品牌论** | 大牌低端款有品牌溢价 | 同价位国产配置更好 |
| **忽视配件** | 琴包、变调夹、备用弦也是成本 | 预算中预留 200 元配件费 |

### 3.2 入门吉他两大类型

| 类型 | 价格区间 | 音色 | 耐用性 | 推荐度 |
|------|---------|------|--------|--------|
| **合板吉他** | 500-1000 元 | 一般，层次感有限 | 坚固耐用 | ⭐⭐ |
| **面单吉他** | 1000-3000 元 | 实木面板，音色明显更好 | 适中 | ⭐⭐⭐⭐⭐ |

> 核心建议：**手感先行**。弦距低、按弦轻松的琴，才能让你坚持学下去。

### 3.3 三步选琴法

```
第一步：确定预算
  800-1500 元 → 入门面单
  1500-3000 元 → 中端面单
  
第二步：选尺寸
  160cm 以下 → 36-38 寸
  160-170cm → 38-40 寸
  170cm 以上 → 40-41 寸

第三步：到店试弹
  感受弦距、指板宽度、琴颈厚度
  同一价位选手感最舒适的
```

## 四、世界顶级名表品牌

### 4.1 品牌分级

| 级别 | 品牌 | 价格区间 |
|------|------|---------|
| **顶级奢华** | 百达翡丽、爱彼、江诗丹顿、朗格、宝珀 | 20 万起 |
| **经典豪华** | 劳力士、欧米茄、万国、积家、宝玑 | 5-30 万 |
| **入门奢华** | 浪琴、泰格豪雅、帝舵 | 1-5 万 |
| **时尚腕表** | 卡地亚、萧邦 | 2-10 万 |

### 4.2 经典系列速查

```
百达翡丽 → Calatrava（正装）、Nautilus（运动）
爱彼      → Royal Oak（皇家橡树）
劳力士    → Submariner（水鬼）、Daytona（迪通拿）
欧米茄    → Speedmaster（超霸）、Seamaster（海马）
万国      → Pilot（飞行员）、Portugieser（葡萄牙）
积家      → Reverso（翻转系列）
宝珀      → Fifty Fathoms（五十噚）
浪琴      → Longines Master（名匠系列）
```

### 4.3 新手入门建议

> 第一次买好表，先明确用途：**正装搭配**选皮带简约款（如积家 Reverso），**日常通勤**选钢带运动款（如劳力士蚝式恒动）。

## 五、选购通用原则

无论买什么，这四条原则通用：

```
1️⃣ 明确预算上限，不要超支
2️⃣ 确定使用场景，对症下药
3️⃣ 到店实际体验，数据不重要，手感最重要
4️⃣ 留预算给配件和维护（镜片护理、吉他配件、手表保养）
```
