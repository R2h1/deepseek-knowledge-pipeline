## 一、科技趋势

### 1.1 6G 技术核心概念

6G 是第六代移动通信技术，预计 2030 年左右商用。

| 代际 | 峰值速率 | 时延 | 商用时间 |
|------|---------|------|---------|
| 4G | 100 Mbps | 50ms | 2010s |
| 5G | 10 Gbps | 1ms | 2020s |
| **6G** | **100 Gbps-1 Tbps** | **0.1ms** | **2030s** |

**6G 的关键特性：**
- **太赫兹频段**：使用 100GHz-10THz 频段，带宽巨大
- **空天地一体化**：卫星+地面基站全覆盖
- **AI 原生**：网络智能自优化
- **感知通信一体化**：网络不仅能通信，还能感知环境

### 1.2 充电宝未来趋势

- 快充普及（65W+ 已成为标配）
- 双向快充（给手机快充，也给充电宝快充）
- 自带线缆（Type-C/Lightning 一体）
- 大容量+轻薄化（氮化镓技术）
- 共享充电宝代理模式 → 收益看地段，核心是"入账率"

## 三、日常生活

### 3.1 买彩票店靠谱吗？

彩票店的盈亏逻辑：

```
收入 = 销售额 × 佣金比例（约 7%-8%）
支出 = 房租 + 人工 + 水电

盈亏平衡点 ≈ 月销售额 5-8 万
```

- 优点：门槛低、稳定
- 缺点：收入上限明显，几乎不可能暴富
- 建议：当成**稳定的小收入来源**，不是致富途径

### 3.2 行李箱容积计算

常见行李箱尺寸的容积参考：

| 尺寸 | 参考容积 | 适合 |
|------|---------|------|
| 20 寸 | ≈ 35L | 短途旅行 1-3 天 |
| 24 寸 | ≈ 60L | 中期旅行 3-7 天 |
| 28 寸 | ≈ 100L | 长途旅行 7 天+ |

### 3.3 买车贷款：无息真的无息吗？

"无息贷款"的四种常见情况：

| 情况 | 真相 |
|------|------|
| 厂家贴息 | 确实无息，但可能限制车型和期限 |
| 手续费代替利息 | 收一笔手续费，实际利率可能不低 |
| 捆绑保险/装潢 | 免了利息，但其他费用上找补回来 |
| 低首付陷阱 | 月供压力大，总利息不少 |

> 买车前算总成本（车价+利息+手续费+保险），别只看月供。

## 四、心理与人际

### 4.1 过度共情

过度共情（Over-empathizing）是指**过度吸收他人的情绪**，导致自己疲惫不堪。

**表现：**
- 看新闻/电影哭得停不下来
- 朋友倾诉完后你比他还累
- 看到别人痛苦，你感同身受甚至更难受

**应对方法：**
```
✅ 建立情绪边界 — 理解别人的痛苦，但不一定要替他承受
✅ 减少信息摄入 — 少看负面新闻
✅ 区分"共情"和"同情" — 共情是理解，同情是怜悯
✅ 保护自己的情绪能量 — 设一个"今天不再想这个问题"的节点
```

### 4.2 聊天话题推荐

和不太熟的人聊天，安全话题清单：

```
🟢 工作/学习（通用安全区）
  "你是做什么的？""最近忙吗？"

🟢 兴趣爱好（容易打开话匣子）
  "周末一般做什么？""有在追什么剧/游戏吗？"

🟢 经历/见闻（分享型话题）
  "最近去过什么好玩的地方吗？""你吃过最好吃的东西是什么？"

🔴 避免（初次聊天）
  收入、年龄、婚姻、政治、宗教
```
