# 基金定投与退休规划指南

> 定投不是一夜暴富的工具，而是普通人参与长期经济增长最稳妥的方式。

---

## 一、基金定投基础

### 1.1 什么是定投？

定投（定期定额投资）就是在固定时间投入固定金额，买入同一只基金。核心优势：**摊平成本、克服择时焦虑、强制储蓄**。

### 1.2 为什么定投适合普通人？

| 一次性买入 | 定投 |
|-----------|------|
| 需要判断"现在是不是低点" | 不管高低点，定期买入 |
| 心理压力大，容易追涨杀跌 | 纪律化操作，减少情绪干扰 |
| 需要一大笔本金 | 每月几百元就能开始 |
| 错过最低点就少赚很多 | 平摊成本，不焦虑择时 |

## 二、定投方案设计

### 2.1 定投金额怎么定？

以一个**月薪 15,000 元**的上班族为例：

```
月薪 15,000
  ├─ 生活支出（50%）：7,500
  ├─ 应急储备（15%）：2,250（存满 3-6 个月支出为止）
  ├─ 定投（10%-20%）：1,500-3,000
  └─ 其他/灵活（15%-25%）
```

**分步执行：**
1. 先存够 3-6 个月应急资金（目标 2.25万-4.5万）
2. 未完成前，减少定投金额，优先存应急金
3. 应急金达标后，将定投金额上调至目标水平

### 2.2 定投频率

| 频率 | 优点 | 缺点 |
|------|------|------|
| **每月一次**（发薪日次日） | 纪律性强，操作简单 | 如果月内波动大，可能买在相对高点 |
| **每两周一次** | 进一步平滑成本 | 操作稍繁琐 |
| **每周一次** | 成本曲线最平滑 | 频繁操作，容易忘记 |

> 对大多数人来说，**每月发薪日自动扣款**是最好的选择——钱还没捂热就去投资了，反而省心。

### 2.3 定投标的选择原则

选择定投标的时，关注以下标准：

```
✅ 长期上涨趋势（宽基指数基金）
✅ 波动大（波动越大，定投摊平效果越好）
✅ 费用低（管理费 + 托管费 < 0.5%/年）
✅ 规模大（> 2 亿，不会清盘）
```

**推荐类型：**
- 国内：沪深 300、中证 500 指数基金
- 海外：纳斯达克 100（QDII 基金）
- 行业：消费、医药等长期增长行业

### 2.4 止盈不止损

定投的核心纪律：

| 操作 | 做法 |
|------|------|
| **下跌时** | 继续买，甚至加码买入（摊低成本） |
| **上涨时** | 按计划继续买 |
| **达到目标收益** | 分批止盈（如收益 20% 卖一半） |
| **市场狂热时** | 可以停止定投，逐步减仓 |

> 定投最忌讳的是：下跌时害怕停投，上涨时追高加仓——恰恰做反了。

## 三、FIRE 退休规划：攒够多少钱可以退休？

### 3.1 核心公式

```
退休金需求 = （每年预期花费 — 社保年收入） × 退休后生活年数
```

### 3.2 不同生活水平的退休目标

按 60 岁退休、活到 85 岁（25 年退休生活）计算：

| 月均花费 | 社保月收入 | 每月缺口 | 需储备总额 |
|---------|-----------|---------|-----------|
| 5,000 | 3,000 | 2,000 | 60 万 |
| 8,000 | 3,500 | 4,500 | 135 万 |
| 10,000 | 4,000 | 6,000 | 180 万 |
| 15,000 | 4,500 | 10,500 | 315 万 |

### 3.3 4% 法则

每年从退休金中提取不超过 **4%** 的金额作为生活开销，本金大概率不会耗尽。

```
所需退休金 = 年开销 ÷ 4%

举例：年开销 12 万 → 需要 300 万退休金
      年开销 20 万 → 需要 500 万退休金
```

> 4% 法则适用于 30 年左右的退休周期。如果提前退休（如 40 岁），建议按 3%-3.5% 计算。

### 3.4 提高退休储备的方法

| 方法 | 效果 |
|------|------|
| **提高储蓄率** | 每月多存 10%，20 年累计效果翻倍 |
| **拉长定投期限** | 越早开始，复利效应越明显 |
| **降低退休预期** | 减少年开销，退休金需求指数级下降 |
| **延迟退休** | 社保月收入更高，储备年限更短 |

## 四、一个完整的定投到退休的计划

```
25 岁 → 开始定投（每月 1,500-3,000）
        ↓
35 岁 → 增加定投金额（收入提高，定投 3,000-5,000）
        ↓
45 岁 → 开始关注资产配置（加入债券、黄金等防御资产）
        ↓
55 岁 → 逐步降低权益仓位，锁定收益
        ↓
60 岁 → 开始提取退休金（按 4% 法则）
```

## 五、常见误区

| 误区 | 真相 |
|------|------|
| "定投一定能赚钱" | 定投赚的是长期增长的钱，选错标的照样亏 |
| "定投就不用管了" | 需要定期检视，太差的基金要及时换 |
| "等有钱了再开始定投" | 100 元也能开始，关键是**开始**而不是金额 |
| "年轻不用考虑退休" | 复利效应下，晚 5 年开始，差距可能是几倍 |

> 种一棵树最好的时间是十年前，其次是现在。定投也一样。
