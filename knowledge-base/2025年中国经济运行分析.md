# 2025年中国经济运行分析

> 2025 年，面对国内外经济环境的复杂变化，中国 GDP 达到 **140.19 万亿元**，同比增长 **5.0%**，实现了"十四五"收官目标。

---

## 一、GDP 总量与增速

| 指标 | 数据 |
|------|------|
| 全年 GDP | 140.19 万亿元 |
| 同比增长 | 5.0% |
| 第一产业 | 93,347 亿元（+3.9%） |
| 第二产业 | 499,653 亿元（+4.5%） |
| 第三产业 | 808,879 亿元（+5.4%） |

**分季度增速：**

```
Q1: +5.4%   Q2: +5.2%   Q3: +4.8%   Q4: +4.5%
→ 呈逐季回落态势，但全年保持在目标区间
```

## 七、核心结论

```
✅ 5.0% 的增长目标圆满完成
✅ 高技术制造业保持两位数增长
✅ 外贸出口韧性较强
⚠️ 房地产投资持续下行
⚠️ 人口负增长趋势延续
⚠️ 消费增速偏缓，通缩压力仍在
⚠️ 青年就业压力仍较大
```

> 2025 年是"十四五"收官之年，经济在复杂环境中保持了基本稳定。2026 年"十五五"开局，关键在于能否在房地产企稳、消费提振、科技创新三条主线上取得突破。
